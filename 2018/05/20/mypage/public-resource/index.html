npm镜像站点

全局安装站点：npm install -g -nrm
使用镜像：nrm use 站点名
查看所有站点：nrm ls
添加镜像地址：nrm add URL 'registry_name'
删除镜像地址：nrm del 'registry_name'
测试镜像地址速度：nrm test ['registry_name']
