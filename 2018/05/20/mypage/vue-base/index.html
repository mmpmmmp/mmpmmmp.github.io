vue基础笔记

环境

安装node.js，具体看公共资源
安装webpack(全局安装只安装一次)：npm install webpack -g
安装vue脚手架(全局安装只安装一次)：npm install vue -cli -g

新建项目

  • 选择一个文件cmd vue init webpack 项目名 创建项目
  • 进入项目文件cmd npm install 安装依赖
  • cmd npm run dev 启动项目
